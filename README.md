📊 Personal Finance Analysis Dashboard

An interactive Excel Dashboard designed to analyze personal financial behavior including income, expenses, savings patterns, and forecasting insights.

📌 Project Objective

The goal of this project is to transform raw financial data into meaningful insights using Excel data analysis and visualization techniques.

This dashboard helps understand how income, expenses, savings, dependents, occupation, and city tiers influence financial behavior.

🧰 Tools & Skills Used

Microsoft Excel

Pivot Tables

Pivot Charts

Slicers

Data Visualization

Forecasting

Data Analysis

📈 Dashboard Features

✔ Income vs Expense Analysis
✔ Savings Contribution Analysis
✔ Expense Distribution by Category
✔ City-wise Income Comparison
✔ Occupation-wise Income Insights
✔ Age-wise Income vs Expenses vs Savings
✔ Dependents Impact on Financial Patterns
✔ Income & Expense Forecasting

🔍 Key Insights

• Tier-2 cities show higher income levels compared to other tiers.
• Expenses increase as the number of dependents grows.
• Groceries and rent contribute to the largest share of expenses.
• Savings trend varies significantly across different age groups.
• Income growth does not always translate into proportional savings.
